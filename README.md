# electron-next-boilerplate
Electron + Next.js boilerplate
Código disponível https://github.com/louisegaiatto/nextJS-electron


### Getting Started
```bash
npm run install:all
```

### Development
```bash
npm run dev
```

### Development isolation
```bash
npm run dev:electron
npm run dev:next
```

### Build
```bash
npm run build:mac
npm run build:windows
npm run build:linux
```
